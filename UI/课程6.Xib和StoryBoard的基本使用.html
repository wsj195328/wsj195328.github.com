1.Xib和StoryBoard介绍

xib的基础知识

  • .xib ＝ nib；
  • .xib的加载：

[[NSBundle mainBundle] loadNibName:@"rowView" ower:nil options:nil];

xib 和storyboard 的不同点

  • storyboard：描述软件界面，大范围，重量级，比较适合描述整个软件的所有界面；
  • xib：描述软件界面，小范围，轻量级，比较适合描述小界面（局部界面）；

xib 和storyboard 的相同点

本质都是转成代码

File‘s owner的使用

  • 在xib文件中设置file‘s owner的class属性
  • 建立File‘s owner 与控件之间的联系
  • 利用代码加载xib，传递owner参数

  • 这时新增的代码文件除了我们之前熟悉的.h和.m文件还多了一个.xib文件

  • 从右下角拖拽控件到中间的View上,并在右上角调整各类控件的样式

  • 通过IBAction关联点击事件

  • 通过Outlet关联属性

  2. - (BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(NSDictionary *)launchOptions {
  3.     // 创建UIWindow

  5.     self.window.backgroundColor = [UIColor whiteColor];
  6.     // 1.第一种方式:没有xib和storyboard

  12.     VC1 *vc1 = [storyboard instantiateInitialViewController];
  13.     // 非根视图控制器 可以使用 storyboardId

  16.     // 3.有指定xib情况下创建
  17.     // 如果通过xib, 会创建xib中描述的veiw做为控制器的veiw

  19.     // 4.有同名xib情况
  20.     // 如果有同名的xib, 会自动找到同名xib中描述的view做为控制器的veiw
  21.     XibVC *vc4 = [[XibVC alloc] init];
  23.     //5.重写控制器的loadveiw方法 如果重写了控制器的loadview方法, 就不会去加载创建同名去掉controller的xib和同名的xib, 而是创建一个空白的veiw做为控制器的veiw
  24.     ViewController *vc5 = [[ViewController alloc] init];
